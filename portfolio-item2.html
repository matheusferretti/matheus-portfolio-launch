HULU Clone

A Replicated HULU App

This HULU Clone was replicated as a test to see how far React.js and a movie database can go. The end project was a satisfyingly responsive web app with lots of movies displayed as well as smooth, dynamic features.

The database utilized for this web app was: TMBD (The Movie Database), and it provides movies as well as movie images, movie descriptions, live movie reviews and constant updates on launching movies.

HULU is a successful company for a reason, but that would not be possible if it wasn't for great programming languages with great features such as React.js, SCSS, and Bootstrap.
